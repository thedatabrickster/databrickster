**Databrickster** is a free, in-depth course that teaches experienced Data
Engineers how to build production-grade AI Agents on Databricks — starting from
the absolute fundamentals of LLMs and ending with enterprise deployment,
governance, and operations.

{/* truncate */}

This "Deep Dives" section will host longer explorations that don't fit the linear
flow of the [core curriculum](/docs/intro) — comparisons, changelogs as the
Databricks AI platform evolves, and extended enterprise case studies.

If you're new here, start with **[the curriculum](/docs/intro)**. It's designed to
be read in order, and every concept is anchored to something you already know from
Data Engineering.
