> Claude Code is an agentic coding assistant that can read your repo, write and run
> code, edit many files, and drive multi-step changes — with you in the loop. This
> subtopic is about using it well as an AI engineer.

## Planned lessons

- **What Claude Code is and how it works** — the agentic loop applied to your codebase (read, plan, edit, run, verify).
- **Scaffolding AI projects** — generating agent skeletons, tools, tests, and Asset Bundles from a prompt.
- **Refactoring & debugging** — using it to trace bugs, write tests, and reproduce failures.
- **Driving multi-step changes safely** — reviewing diffs, keeping a human in the loop, and knowing where it needs guardrails.
- **Claude Code + Databricks via MCP** — connecting to Unity Catalog functions, Vector Search, and Genie so the assistant can use governed tools.
- **Good habits** — prompting, verification, and when to trust vs check the agent.
